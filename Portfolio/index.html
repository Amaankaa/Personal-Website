My Projects

Library Management System

This Library Management System is a web-based application designed to streamline book tracking, borrowing, and user management. Built with a focus on simplicity and efficiency, it allows admins to manage inventory, track issued books, and maintain student records—all from a clean, intuitive interface. Whether it’s checking availability or managing returns, this LMS handles it all with ease.

Bank Management System

This Bank Management System is a console-based C++ project that simulates basic banking operations like creating accounts, depositing, withdrawing, and checking balances. Designed with a file-handling backend, it securely stores user data and ensures smooth transaction processing. It’s a solid demonstration of object-oriented programming, file I/O, and real-world system logic implemented in C++.

YouTube homepage

This project is a front-end recreation of YouTube's homepage, built using HTML and CSS. It focuses on replicating the layout, styling, and responsiveness of the original design to sharpen UI skills and understand how modern video platforms structure their interfaces.
